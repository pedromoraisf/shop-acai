## Instalação e usagem
### 🐳 Pelo Docker
```
cd api && yarn
```
```
cd frontend && yarn
```
```
cd docker && docker-compose up --build
```

### 🤖💅🏽 Pelas Stacks
```
cd api && yarn && yarn dev
```
```
cd frontend && yarn && yarn dev
```
